project requirement:

1) nodeJS from: https://nodejs.dev/
2) Bootstrap jquery
3) angular/cli
4) angular/material
5) angular/cdk

to run the project:

1) install angular/cli, angular/material & angular/cdk in your device by running this command
	$ npm install -g @angular/cli
	$ npm install -g @angular/material
	$ npm install -g @angular/cdk

2) check your angular version
	$ ng --version

3) CD into this project folder and install Bootstrap and jquery
	$ cd NatroxForm
	$ npm install --save bootstrap jquery

4) install firebase module by running this command
	$ npm install -g firebase-tools

4) run the project
	$ ng serve
